<h1>Spotify Clone</h1>
    <p><strong>Frontend Project using HTML, CSS, and JavaScript</strong></p>

    <h2>Overview</h2>
    <p>
        This project is a Spotify UI Clone built using HTML, CSS, and JavaScript.
        It replicates the basic look and feel of Spotify’s web player and is intended
        for learning, practice, and portfolio showcase.
    </p>

    <h2>Technologies Used</h2>
    <ul>
        <li>HTML5 – Page structure</li>
        <li>CSS3 – Styling and layout</li>
        <li>JavaScript (Vanilla) – Interactivity and audio control</li>
    </ul>

    <h2>Features</h2>
    <ul>
        <li>Spotify-like user interface</li>
        <li>Play / Pause music</li>
        <li>Song list display</li>
        <li>Progress bar control</li>
        <li>Basic audio player functionality</li>
        <li>Responsive layout</li>
    </ul>

    <h2>Project Structure</h2>
    <pre>
spotify-clone/
│
├── index.html
├── style.css
├── script.js
├── assets/ (songs / images if any)
└── README.html
    </pre>

    <h2>How to Run the Project</h2>

    <h3>Method 1: Open Directly</h3>
    <ol>
        <li>Download or clone the repository</li>
        <li>Open <code>index.html</code> in any modern web browser</li>
        <li>The application will run locally</li>
    </ol>

    <h3>Method 2: Using Live Server</h3>
    <ol>
        <li>Open the project in VS Code</li>
        <li>Install the Live Server extension</li>
        <li>Right-click on <code>index.html</code> and select "Open with Live Server"</li>
    </ol>
